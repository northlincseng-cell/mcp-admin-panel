# archive — previous versions

this folder preserves all earlier iterations of the mcp admin panel. nothing gets deleted.

## v1-static-html/
the original static html panel (2,876 lines). pure frontend, no database.
built as a design prototype to establish the dark green brand aesthetic.

## v2-fastapi-backend/
intermediate fastapi + sqlite backend (`api_server.py`).
wired to the static html. superseded by the full react + express rebuild.

## docs/
- `gs-master-control-panel-spec.pdf` — 22-page specification document.
  note: some entity relationships in this doc are known to be incorrect (parked for rework).
